# Softchoice PM Hub

A lightweight personal project management tool for a Google Cloud Solution Consultant at Softchoice.

## Features
- Track active projects with stage, client/team, milestone date, and notes.
- Capture weekly tasks with week-start date and priority.
- Track monthly outcomes and status (planned, in progress, achieved, deferred).
- Save profile details and all records in browser `localStorage`.

## Run locally
```bash
python3 -m http.server 8000
```
Open `http://localhost:8000` in your browser.
